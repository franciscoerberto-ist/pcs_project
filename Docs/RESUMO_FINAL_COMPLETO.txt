📋 REQUISITOS TÉCNICOS

Obrigatório:
  ✓ Java 17+, Spring Boot 3.x, Maven/Gradle
  ✓ JPA/Hibernate, H2/PostgreSQL
  ✓ Git + GitHub

Esperado (alta nota):
  ✓ Bean Validation + Custom Validators
  ✓ @ControllerAdvice Global
  ✓ Swagger
  ✓ JUnit 5 + Mockito (>70% cobertura)
  ✓ DTOs
  ✓ Paginação
  ✓ README completo

Bônus (nota máxima):
  ✓ Spring Security
  ✓ Cache (@Cacheable)
  ✓ Auditoria (@CreationTimestamp)
  ✓ Soft Delete
  ✓ PATCH endpoints
  ✓ Busca avançada
  ✓ GitHub Actions (CI/CD)
